# FocusLock 🚀

### Goal-based productivity app that blocks distracting apps until your task is completed.

FocusLock helps students and professionals stay focused by blocking distracting applications while they work on their goals.
Instead of relying on willpower, FocusLock creates a distraction-free environment so you can finish what you start.

---

## ✨ Features

* 🎯 **Goal-based focus sessions**
  Set a task, start time, and deadline.

* 🔒 **Automatic app blocking**
  Social media and other distracting apps are blocked during the focus session.

* ⏱ **Focus timer**
  Tracks your progress until the task is completed.

* 📊 **Productivity tracking**
  Helps you understand your work habits.

* 📱 **Minimal UI**
  Clean interface designed for quick goal setup.

## 🧠 How It Works

1. User sets a **goal or task**
2. User selects **start time and deadline**
3. FocusLock activates **focus mode**
4. Selected distracting apps are **blocked**
5. Session ends only when the task is completed or time ends

This approach helps reduce procrastination and improve productivity.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Roushanverse/focuslock.git
```

Go to project directory:

```bash
cd focuslock
```

Install dependencies:

```bash
flutter pub get
```

Run the app:

```bash
flutter run
```
